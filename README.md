# Hardware-and-Wireless-security

Questa repository contiene i progetti dei 2 laboratori realizzati per la materia **Wireless and device to device communication**.

## Laboratorio 1: Analyzing Goodput in Different Network Scenarios : Ethernet vs Wi-Fi
Il primo laboratorio si concentra su esplorare la performance di una rete in 3 scenari diversi per TCP e UDP: due computer collegati tramite cavo ethernet, due computer collegati tramite rete WiFi e lo scenario misto



## Laboratorio 2: Accuracy and Robustness Analysis of Raw GNSS Data in Diversified Scenarios
Il secondo laboratorio si concentra sull'analisi dei dati GNSS in uno scenario di spoofing, in uno scenario di buona connessione e cattiva connessione e gli effetti di schermatura della carta di alluminio
